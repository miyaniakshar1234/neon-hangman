# Security Policy

## Supported Versions

We are committed to keeping Neon Hangman secure. Currently, only the latest version of the project is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it privately. **Do not create a public issue for security-related reports.**

### How to report

Please report any security vulnerabilities by opening a private security advisory on GitHub or by contacting the project maintainer directly.

We will acknowledge your report within 48 hours and provide a timeline for a resolution. We kindly ask for responsible disclosure, allowing us time to patch the vulnerability before it is made public.

Thank you for helping keep this project safe!
